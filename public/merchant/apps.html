Integration Guide
  1. Register your app to get an API key
  2. Use x-api-key header in API requests
  3. Create a payment session via POST /create-payment-session
  4. Redirect customer to the checkout_url
  5. Customer selects method and pays
  6. Receive webhook at your callback URL
  7. Verify status via GET /payment-status
